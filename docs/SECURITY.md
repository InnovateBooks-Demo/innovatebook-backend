\# Security Policy



\## Reporting a Vulnerability

If you discover a security issue, \*\*do not open a public issue\*\*.



Please report responsibly by contacting:

\- Email: security@innovatebooks.in



\## Guidelines

\- Do not share secrets in issues or PRs

\- Do not commit credentials or tokens

\- Use environment variables for all secrets



Thank you for helping keep InnovateBooks secure.
